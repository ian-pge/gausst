# Gausst

Unified environment for Depth-Anything-3, gsplat, sam3, and sfm.

## Installation

### 1. Install Pixi
If you haven't installed Pixi yet, run the following command:

```bash
curl -fsSL https://pixi.sh/install.sh | bash
```

### 2. Install Project Dependencies
Navigate to the project directory and install the environment:

```bash
pixi install
```

### 3. Verify Installation
You can verify that everything is set up correctly by running the import test:

```bash
pixi run test-imports
```
